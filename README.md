# Rails Graphql

A Ruby api with Graphql and PostgreSQL implementation

## Setup

### Gems
```shell
bundle install
```
### Develpment env
```shell
rake dev:setup
```
### Start server
```shell
rails s
```
### GraphiQL

[localhost:3000/graphiql](http://localhost:3000/graphiql)
